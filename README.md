# scss boilerplate

install package
```bash
yarn add bo_style
```

## usage
1. create config like this [default-config](https://github.com/shibnev/bo-style/blob/main/scss/default-config.scss)
2. import folder and your config

```scss
@import './scss';
@import './your-config.scss';
```
